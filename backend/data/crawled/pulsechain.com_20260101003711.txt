Prepare Your Wallet
Wallets, such as Internet Money, MetaMask, and Brave Wallet , interact with PulseChain.


Set Up Wallet
More Details
Install a wallet (Internet Money, MetaMask, or Brave Wallet)
Find the Switch Network button in your wallet software and search for a Add Custom Network. Enter the network settings below.

After installing, some wallets will let you add PulseChain by clicking here.
Find the Custom Network button and add the following settings:
Network Name: PulseChain
RPC URL: https://rpc.pulsechain.com
Chain ID: 369
Currency Symbol: PLS
Block Explorer URL: https://ipfs.scan.pulsechain.com
Buy PLS
PLS is required for all transactions and can be bought directly via bank-to-crypto platforms, such as 0xCoast or Rampnow.

Existing crypto users can use LibertySwap, Gibs Finance, or other methods.


Buy PLS Today
More Details

PLS is the native gas token of PulseChain and is required for all transactions, including sends, swaps, and interacting with contracts.
