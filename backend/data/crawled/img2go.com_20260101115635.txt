They store photo and picture information for each pixel in the image. That is why raster images cannot be scaled up without losing quality and detail. They can be compressed in different ways, which affects the file size. Some raster images support transparency and even animation. Common vector image formats Vector images are very common in graphic design and are often used as web graphics. The image data in a vector image is calculated using mathematical formulas, which allows the image to be enlarged without losing quality. However, not all image viewers can open vector images. It is very easy to edit and manipulate vector images with the right software.
