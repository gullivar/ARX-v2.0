Un VPS (Virtual Private Server) es una combinación entre un servidor dedicado y un hosting web compartido. En el hosting compartido lo que se comparte es un servidor y sus recursos con otros sitios web. A pesar de compartir algunos recursos, el software funciona como si fueran múltiples servidores diferentes. Le asigna a cada usuario uno. Utilizando un VPS tenés el poder administrativo de una configuración de host dedicada, esto incluye la capacidad de personalizar el servidor para que se adapte a cada webmaster.
						Si querés mantener costos bajos pero al mismo tiempo necesitás más potencia que la ofrecida por un servidor compartido, un servidor VPS es la mejor opción para vos. ¿Cuánto tiempo tarda el alta del hosting? El ancho de banda puede medirse por la cantidad de datos que se transmiten en una conexión a Internet. Funciona de la siguiente manera: Cuando alguien quiere visitar tu página web, se envía una solicitud al servidor. A su vez el servidor responde y le muestra la información que hay en la página al usuario. Mientras más personas entren a tu sitio al mismo tiempo más ancho de banda vas a consumir, hasta llegar a un punto en el que se agotará la capacidad del ancho de banda. Esto trae como consecuencia que la web funcione lenta o incluso que deje de estar online. ¿Qué es una base de datos MySQL? MySQL es una herramienta de administración de bases de datos de código abierto que emplea lenguaje SQL para acceder, añadir y gestionar el contenido de una base de datos. 
						Se utiliza para almacenar y organizar información, de forma rápida, sencilla y simple. La mayoría de los CMS (sistemas de gestión de contenido como por ejemplo WordPress) modernos lo utilizan para almacenar y manipular datos. 
						Es decir, al realizarse una modificación en el CMS se registra en la base de datos MySQL. ¿Qué es un certificado SSL? Es un certificado que cifra la información enviada a través de la web (por ejemplo números de tarjetas de crédito, u otros datos personales) y la encripta para que ésta no se pueda usar de forma no autorizada. Es un protocolo que proporciona un canal seguro entre dos dispositivos que operan a través de la Internet. Este nivel de seguridad impide que la información pueda ser robada por parte de un tercero. Por lo tanto, contar con un certificado SSL reduce el riesgo de que la página web pueda ser vulnerada e incrementa la confianza de los clientes, usuarios y visitantes ya que les asegura que su información estará protegida, es decir, que nadie más puede visualizarla.
						Cuando un sitio web está protegido por un certificado SSL, en la barra de dirección, delante del dominio o URL vas a ver las siglas HTTPS (Hyper Text Transfer Protocol Secure en inglés o Protocolo Seguro de Transferencia de Hipertexto).
