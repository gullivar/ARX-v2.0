Build and Run Containers Using with nerdctl or the Docker CLI you can build, run, and otherwise work with images and containers. How it works? Rancher Desktop is an Electron-based application that wraps other tools while providing a simple user experience. On macOS and Linux, Rancher Desktop uses a virtual machine to run containerd or Docker and Kubernetes. For Windows systems, it leverages Windows Subsystem for Linux v2. To get started, simply download and run the application. How to get involved Rancher Desktop is an open source project hosted on GitHub. Bugs and features are tracked through issues and new code is reviewed through pull requests. Issues Pull Request How to get support Found an issue or have a problem?
