libravatar :: federated avatar hosting service libravatar freeing the web one face at a time Login Sign up The open avatar service libravatar is a service which delivers your avatar (profile picture) to other websites. If you create an account with us, your photo could start popping up next to forum posts or blog comments on any site where you left your email address. Read more... Federated Open Source Service This service is powered by Free and Open Source software called ivatar . With this software you can also run your own instance and serve avatars yourself. Simple API for Developers Application developers can easily add support for this service using our simple API or one of the libraries and plugins available for a number of platforms and languages.
