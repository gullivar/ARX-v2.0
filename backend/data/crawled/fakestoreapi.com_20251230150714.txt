Resources

FakeStoreAPI provides the 4 essential resources commonly used in e-commerce apps. Use them to build realistic flows for your frontend or backend prototypes.

Products
List of items with title, price, category, description and image
20 products
Cart
Includes full user profiles (name, email, address, etc.)
20 carts
Users
Simulates shopping cart logic: user + list of product IDs + quantity
10 users
Login Token
Returns a JWT token for testing authentication
JWT token
