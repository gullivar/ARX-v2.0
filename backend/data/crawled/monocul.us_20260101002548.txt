What kind of alcohol would sysadmin like? > root beer What is Monoculus? Monocul.us was born between 2014 and 2015 with the mission of providing non-profit network services focused on freedom and privacy .
      Initially founded to run game and voice servers, in 2015 the name Monoculus was chosen as a tribute to the Team Fortress 2 Halloween event monster. ...Everyone thinks I'm just a one-eyed bloody monster... Monoculus aims to be the first node of a server infrastructure closer to the true spirit of the network : modular, collective, scalable , and designed to integrate additional nodesmoving away from the traditional monolithic model of a single machine dispensing services. What We Offer Game server hosting Community projects Voice servers (VoIP) File archiving for lost media & controversial content Experimental network projects And if times get tough, we'll send out monkeys to build WiFi/Radio bridges . sysop, andrea@monocul.us Powered by Debian, NGiNX and Jekyll (and the fantastic hackss framerwork )
