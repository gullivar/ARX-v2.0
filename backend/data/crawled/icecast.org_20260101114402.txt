Page navigation Docs FAQ Download Apps Ezstream IceS Streams Contributing Contact About Icecast 2 Icecast is a streaming media (audio/video) server which currently supports
	Ogg (Vorbis and Theora), Opus, WebM and MP3 streams. It can be used to create an Internet radio station or a privately
	running jukebox and many things in between.
	It is very versatile in that new formats can be added
	relatively easily and supports open standards for communication and
	interaction. Icecast is distributed under the GNU GPL, version 2. Latest News Icecast Release 2.5.0-rc2 We are pleased to announce Icecast 2.5.0-rc2 (2.4.999.2).
