Preguntas Frecuentes sobre Hosting y Alojamiento Web ¿Qué es el servicio de hosting o alojamiento web? El servicio de hosting (también llamado alojamiento web) consiste en arrendar un espacio dentro de un servidor ubicado en un datacenter conectado a internet las 24 horas. En este espacio se almacena tu sitio web, correos corporativos, bases de datos y cualquier archivo necesario para que tu página funcione. Para utilizar hosting, es indispensable contar con un dominio, ya que es la forma de identificar un sitio. ¿Cómo se utiliza un servicio de web hosting? Para comenzar, el cliente debe tener su propio dominio activo. Una vez contratado el servicio de web hosting en Ninjahosting y con los DNS apuntando a nuestros servidores, tu cuenta estará lista para usarse. Puedes subir tu sitio web mediante: •	FTP, utilizando cualquier cliente compatible, o •	Directamente desde el panel de control, donde también puedes crear y administrar tus cuentas de correo personalizadas. Para quién está orientado el servicio de hosting? Nuestros planes de hosting básico económicos están diseñados para: •	Nuevos emprendimientos •	Sitios web pequeños •	Proyectos informativos •	Personas naturales que buscan un alojamiento web económico Si necesitas mayor rendimiento, contamos con planes avanzados ideales para sitios exigentes como tiendas online o proyectos hechos en WordPress, que requieren más velocidad y recursos de servidor. ¿Dónde puedo comprar un dominio? Existen diversas extensiones de dominio. En Chile, la extensión .cl se puede adquirir directamente en nic.cl.
Otras extensiones de dominios globales como .com, .net o .org se registran mediante proveedores internacionales como enom.com. También puedes registrar tu dominio directamente en Ninjahosting, junto con tu servicio de alojamiento web. ¿Cómo comenzar a utilizar mi plan de web hosting? Si contratas un plan de hosting + dominio con Ninjahosting, habilitaremos el dominio una vez confirmado el pago. Aunque el web hosting queda activo inmediatamente, el dominio se propaga solo después de la confirmación. Si ya tienes un dominio registrado en otra empresa, solo debes contratar el servicio de webhosting y actualizar los DNS que recibirás en tu correo de bienvenida. • Para dominios .cl, se cambian en nic.cl • Para otras extensiones, se modifican en el panel del registrador correspondiente. ¿Cómo elegir el plan de hosting adecuado?
