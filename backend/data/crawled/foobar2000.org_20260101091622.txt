Main features Plays all mainstream audio formats out of the box. Additional decoder components enable even more formats. Gapless playback. Stream music on your local network; foobar2000 can be used to control network audio players or foobar2000 running on another device. Easily customizable user interface layout on Windows and macOS . Advanced tagging capabilities . Support for ripping Audio CDs as well as transcoding all supported audio formats. Full ReplayGain support. Customizable keyboard shortcuts. Many built-in DSP effects, additional available through DSP components ; can load VST effects via VST adapter on Windows, Audio Units on macOS. Internet radio support, including integration with Radio Browser . Open component architecture allowing third-party developers to extend functionality of the player - browse add-on components Available on: Windows , macOS , Android and iOS . Windows version works on Linux via Wine.
