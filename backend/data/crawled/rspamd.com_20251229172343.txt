Key Features
Powerful Open Source Spam Filtering

Rspamd provides state-of-the-art spam protection with a focus on performance, flexibility and ease of use. Leveraging AI, machine learning, and advanced analytics for comprehensive email security.

Advanced Protection

Multi-layered filtering including SPF, DKIM, DMARC validation, URL black lists, pattern detection, and machine learning techniques.

AI & LLM Integration

Leverage Large Language Models and AI for advanced spam detection, content analysis, and intelligent filtering decisions.

High Performance

Written in C with event-driven architecture, achieving processing rates of hundreds of messages per second using modest hardware.

Advanced Analytics

Powerful data analysis with ClickHouse and Elasticsearch integration for deep insights into email patterns and threats.

Easy Integration

Works with major MTAs including Postfix, Exim, Sendmail and has native support in many mail server solutions.

Web Interface

User-friendly web interface for monitoring, configuration and checking message status in real time.

Machine Learning

Advanced neural networks and statistical analysis for adaptive spam detection that improves over time.

Extensible Architecture

Modular design with plugin support and Lua scripting for creating custom rules and integrations.

Comprehensive Documentation

Detailed user guides, configuration examples, and development resources to help you get the most out of Rspamd.

Open Source
Community-Driven Development

Rspamd is 100% free and open source software released under the Apache 2.0 license. Join our community of contributors and users helping to make email safer for everyone.

Powerful Core Features

Benefit from years of development with comprehensive spam filtering capabilities, including content analysis, RBLs, SPF, DKIM, DMARC, and much more.

ClickHouse & Redis Integration

Store and analyze large volumes of email metadata using ClickHouse integration, and benefit from high-performance Redis backend for caching and data storage.

Web Interface & Advanced Tooling

Manage your installation through an intuitive web interface and leverage advanced tooling for configuration, monitoring, and fine-tuning of spam filtering rules.

Antivirus Engine Support

Seamlessly integrate with popular antivirus engines to enhance email security and detect malicious attachments before they reach users' inboxes.

Personal Settings & Customization

Tailor Rspamd to your specific needs with extensive personalization options, custom rules, and per-user settings for optimal filtering performance.
