iTerm2 - macOS Terminal Replacement Menu Home News Features FAQ Documentation Downloads What is iTerm2? iTerm2 is a replacement for Terminal and the successor to iTerm. It works on Macs with macOS 10.14 or newer. iTerm2 brings the terminal into the modern age with features you never knew you always wanted. Why Do I Want It? Check out the impressive features and screenshots . If you spend a lot of time in a terminal, then you'll appreciate all the little things that add up to a lot. It is free software and you can find the source code on Github . How Do I Use It? Try the FAQ or the documentation . Got problems or ideas?
