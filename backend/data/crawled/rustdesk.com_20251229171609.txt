Simple self-hosting installation

Step 1: Install Docker

bash <(wget -qO- https://get.docker.com)

Step 2: Download compose.yml

wget rustdesk.com/oss.yml -O compose.yml

or

wget rustdesk.com/pro.yml -O compose.yml

Step 3: Compose

docker compose up -d

Ready!
