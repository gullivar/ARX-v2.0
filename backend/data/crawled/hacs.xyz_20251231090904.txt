HACS HACS Home Initializing search hacs/integration Home Use Publish Contribute FAQ Help The Home Assistant Community Store ( HACS ) is a custom integration that provides a UI to manage custom elements in Home Assistant . What HACS can do: Help you discover new custom elements . Help you download new custom elements . Manage (update/remove) custom elements . Publish your own custom element repository and create shortcuts to repositories or issue trackers.
