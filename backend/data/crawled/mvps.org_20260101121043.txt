Welcome to the MVPs.org home page! This is your jumping off point to a number of interesting offerings being
      provided for you by a few folks associated with the Microsoft Most Valuable Professional (MVP) program. Plan to check this page every
      so often as new subwebs and subdomains are bound to show up from time to time.
      This home page has been somewhat organized to categorize our offerings, in
      the hope of narrowing down your quest for information. The sites that are described
      below are those hosted by this domain. The MVP Links page points to all known MVP sites. Development Applications Platforms Site Info Application Development Alex Angelopoulos' Windows Scripting Site Resources for Windows scripters, focusing primarily (but not exclusively) on WSH.  Topics include extension of scripting with COM, tools for scripters, performance analysis, numerical scripting, scripting and programming interoperability, scripting philosophy, and the ever-popular Rube Goldberg memorial scripting page commemorating bizarre scripting solutions. Robert Dunlop's X-Zone A useful resource for game developers and others using the DirectX SDK for multimedia development. Johnson's Slightly Tilted Software All of the code from articles that L.J. has done for VBPJ over the years. Mainly NT administrative-type programs and utilities, but many are useful for anyone who uses NT and Visual Basic. A sampling of WMI and ADSI programming is also included, along with photos of other MVPs and well-known VB personalities from VBits.
